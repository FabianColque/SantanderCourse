Web Server

eh um software por receber, processar e enviar solicitacoes de paginas web 
para os clientes que as requisitam.

Funciona atraves de um protocolo de comunicacao. 
Quando um cliente(navegador web) faz uma solicitacao para o server web

Populares
Apache, BGINX, IIS

Apache
eh um servidor web gratuito e de codigo aberto que foi lancado em 1995
Seu nome eh HTTPD, Apache eh o projeto responsavel pelo seu desenvolvimento.
Suporta widows linux e unix

Linguagens de programacao para Web

HTML
    Nao eh linguagem de programacao
    Hipertext markup language
    Organiza a distribuicao de elementos
    
CSS
    Cascading style sheets
    Linguagem de estilo
    apariencia visual das paginas web
    
    
Javascript
    Client side rendering -> que o browser que executa o codigo
    linguagem interpretada, que nao precisa ser compilada
    
        
PHP
    Hypertext preprocessor
    eh uma linguagem de programacao utilizada pra criar aplicacoes web dinamicas. 
    Ele permite nteragir com bnaco de daods,. enviar emails, criar formularios e outroas coisas
    Eh uma linguagem server side rendering, ou seja todo codigo eh executado no servidor e apenas o
    resultado eh enviado ao navegador do cliente
    O Php possui uma vasta comunidade de desenvolvedores e suporte de diversas plataformas tornando-a uma opcao
    popular para o dessenvolvimento de prohetos web de diferentes tipos e tamanhos.
